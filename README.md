# Projeto Multidisciplinar – Front-end

Front-end desenvolvido com Django, utilizando HTML e CSS para a construção
das interfaces do sistema de balcão.

Este projeto faz parte de um Trabalho de Conclusão de Curso (TCC) e foi
desenvolvido com foco na integração entre interface e back-end,
priorizando funcionalidade e clareza.

## Tecnologias utilizadas
- Django
- HTML
- CSS

## Funcionalidades
- Telas para interação com o sistema de balcão
- Páginas dinâmicas integradas ao back-end em Django
- Estruturação de templates para organização das telas
- Estilização das páginas utilizando CSS

## Organização do projeto
O front-end foi desenvolvido dentro do próprio ecossistema do Django,
mantendo a separação entre lógica de apresentação (templates)
e lógica de negócio (back-end).

A estrutura do projeto prioriza simplicidade, organização
e integração direta com o servidor.

## Observações
O foco deste projeto está na funcionalidade e integração com o back-end.
Não foram utilizados frameworks front-end externos ou bibliotecas adicionais,
apenas Django, HTML e CSS.
